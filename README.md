Beth and Ian's Wedding Site
===========================

This is the wedding site for Beth Harmon and Ian Li. It will be hosted at
http://harmonli.com


Development
-----------

Install Jekyll (http://jekyllrb.com).

In this directory, run the following:

    > jekyll serve
